Citizen ID Command

Custom Encryption

A simple FiveM script that adds the /cid command, allowing players to view their Citizen ID via an ox_lib notification.
Uses qbx_core to retrieve player data and includes basic error handling if the Citizen ID is unavailable.

Drag and drop into your resources 

In your server cfg "ensure zeraphis-cid"

Requirements: qbx_core, ox_lib
